# rabbit_theme-sample

プレゼンツールRabbitを使ってテーマを自作した時のサンプルです。  
公式サイト：[Rabbit - はじめに](https://rabbit-shocker.org/ja/)

　  
# 動作確認環境

- Mac OS X 10.11.6
- Ruby 2.4.0p0
 - rbenvを使ってインストール
- rbenv 1.1.0
- rabbit 2.2.0
 - Bundlerを使ってインストール
- bundler 1.14.4
